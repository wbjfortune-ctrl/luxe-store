# 🛍️ Luxe Store

A mini e-commerce store built with pure HTML, CSS & JavaScript — no frameworks, no dependencies.

## Features

- Product grid with category filters (All, Tech, Fashion, Home, Beauty)
- Add to cart with quantity controls
- Sliding cart sidebar with total price
- Toast notifications on add
- Fully responsive layout

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Run Locally

Just open `index.html` in your browser — no setup needed!

## Deploy Live (Free)

1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click **New Project** → Import `luxe-store`
4. Click **Deploy** — done! 🚀
